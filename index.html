About

This website is a project to compare various sorting algorithms. While some algorithms have better average efficiency, this doesn't neccessarily imply efficiency when data is not distributed uniformingly randomly. Hence, by simulating uploaded data from a file, we can easily view the times taken to process the list with the most popular sorting algorithms used today.

How To Use

Upload the data in a .txt format. Currently, only positive integers are supported. Don't worry if the uploaded data isn't in a nice format, as this is filtered before results are produced.
